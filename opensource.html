ACE-RTL: Agentic Context Evolution

项目概述

  • 论文标题："When Agentic Context Evolution Meets RTL-Specialized LLMs"
  • 发布时间：2025年
  • 来源：arXiv (2602.10218)
  • 类型：学术研究

ACE-RTL 是一个将 Agentic Context Evolution（代理式上下文演化）与 RTL 专用大语言模型相结合的框架，通过智能代理机制提升硬件设计自动化水平。

核心能力

  • RTL 专用 LLM：针对 Verilog/SystemVerilog 优化的语言模型
  • 上下文演化：动态更新和优化设计上下文
  • 代理式工作流：多步推理和工具调用能力
  • 自动代码生成：从自然语言规格生成 RTL 代码

Saarthi: 首个 AI 形式验证工程师

项目概述

  • 论文标题："Saarthi: The First AI Formal Verification Engineer"
  • 发布时间：2025年
  • 来源：arXiv (2502.16662)
  • 开发者：Infineon Technologies（英飞凌）
  • 类型：工业研究

Saarthi 是全球首个完全自主的 AI 形式验证工程师，能够端到端验证 RTL 设计，从简单的计数器和 FSM 到复杂的 FIFO 结构。

核心能力

  • 全自主验证：无需人工干预的形式化验证
  • 多 Agent 协作：多个智能代理协同工作
  • 端到端流程：从断言生成到验证闭合
  • 复杂结构支持：支持 FIFO、FSM、计数器等
  • 覆盖率闭合：基于 AI 的覆盖率达到成

VerilogEval: RTL 生成质量基准测试

项目概述

  • 论文标题："Evaluating Large Language Models for Verilog Code Generation"
  • 发布时间：2023年（v2: 2024年）
  • 来源：arXiv (2309.07544)
  • 开发者：NVIDIA Labs
  • 类型：开源基准测试

VerilogEval 是由 NVIDIA Labs 开发的 Verilog 代码生成评估基准，用于客观评估大语言模型在硬件设计代码生成任务上的表现。

核心特性

  • 数据集规模：156 个问题
  • 来源：HDLBits（Verilog 教学网站）
  • 任务类型：
    • 代码补全（Code Completion）
    • 规格到 RTL 生成（Specification-to-RTL）
  • 评估指标：功能正确性、语法正确性
  • 开源可用：GitHub 仓库公开

相关基准与扩展

  • VerilogEval v2：增强版基准，ACM 发布
  • RTLCoder-Data：18 个设计的开源数据集，包含规格、信号定义和正确 RTL 代码
  • Comprehensive Verilog Design Problems：下一代代理导向基准，支持生成、验证和相关任务
