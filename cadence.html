核心 Agent 能力

  • RTL Code Agent：自然语言到 RTL 代码的自动生成，支持 Verilog/SystemVerilog
  • Design Upgrade Agent：设计升级自动化，支持 IP 迁移和工艺节点转换
  • RTL Optimization Agent：代码级 PPA 优化建议和自动重构
  • SoC Agent：系统级集成自动化，处理多模块协作设计
