公司背景

  • 公司名称：Siemens Digital Industries Software
  • 母公司：Siemens AG（德国西门子）
  • 成立时间：Siemens EDA 前身为 Mentor Graphics（1981年）
  • 收购时间：2017年 Siemens 收购 Mentor Graphics
  • 总部位置：美国俄勒冈州威尔逊维尔
  • 行业地位：全球 EDA 三巨头之一

核心 Agent 能力

  • RTL Code Agent：自主 RTL 代码生成、代码检查（Linting）、时钟域交叉分析（CDC）
  • Verification Planning Agent：验证计划创建、执行跟踪、覆盖率闭合
  • Debug Agent：加速调试工作流，自动问题定位和分析
  • Human-Centered AI：人机协作模式，工程师保持控制权
